A simple library system made in C language using the program CodeBlocks, the system does the following operations in the CMD console:
1. adds books with their names, bin, and price to a text file
2. removes books using the bin number
3. Search a book from the text file
4. Print all the books and their data from the text file to console
5. uploads and updates the data from and to the text file
This was the project given to students who took the course "COMPUTER AND PROGRAMMING,s erial:comp133" in Birzeit university , /spring of 2025
